# AI_Study_Chatbot
An AI-powered study chatbot developed in Python using functions and dictionaries. The chatbot responds to user queries, provides study-related assistance, and demonstrates the use of structured data management and modular programming.

## Features
- Interactive question-and-answer chatbot.
- Uses Python dictionaries to store questions and responses.
- Functions are used to organize chatbot operations and improve code readability.
- Easy to customize by adding new questions and answers.

## Technologies Used

-  Python
- Functions
- Dictionaries
- Conditional Statements

## Learning Outcomes
- Understanding the use of dictionaries for data storage and retrieval.
- Implementing functions for modular programming.
- Building a basic rule-based AI chatbot.
- Learning user input handling and response generation.
